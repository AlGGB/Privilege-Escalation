# Privilege-Escalation
This Script are create with educational porpuse, and to learn and repeat in controlled environmentcontrolle. 
This script is designed to automate the process of finding potential privilege escalation vectors during security audits or CTF challenges. It checks for common misconfigurations, weak permissions, and exploitable binaries.

Key Features
System Info: Kernel version, hostname, and OS details.

Users & Groups: Enumerates current user privileges and sensitive files.

SUDO & SUID: Finds binaries with SUID bits and misconfigured sudo permissions.

Network: Lists open ports and internal services.

Cron Jobs: Identifies writable scheduled tasks.
